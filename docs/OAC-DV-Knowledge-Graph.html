📊 What is Oracle Analytics Data Visualization?

OAC Data Visualization (DV) is the self-service analytics layer of Oracle Analytics Cloud. It lets you connect to data, shape it, and explore it visually — without writing code.

You bring data together into datasets, then build workbooks made of canvases and visualizations — 40+ chart types created by dragging columns. You make them interactive with filters, parameters and data actions; let the platform surface patterns automatically with Auto Insights and Explain; and ask questions in plain English using the built-in AI Assistant and AI Agents. The result is a fast path from raw data to a decision-ready story.

👥 Who works in OAC — the personas

OAC serves three kinds of users. This workshop is built mainly for the first two; the third keeps the platform running behind the scenes.

👀

Business Users — Consumers

🛠️

Power Users — Content Creators

🛡️

Provisions the service, manages users, security and connections, and keeps the platform healthy — the persona working behind the scenes.

🤖 Day 2 — OAC AI features & the feedback pipeline

Day 2 builds on the same workbook to explore OAC's AI features and, just as importantly, how to make them better over time.

1

Enable the AI features

Turn on and configure the AI Assistant and AI Agents so they're ready to query your data conversationally.

2

Use them for effective responses

Ask the same retention questions in plain English and learn the prompting patterns that produce accurate, useful answers.

3

Enable the feedback pipeline

Review the generated Logical SQL (LSQL) behind each answer to see exactly how the AI interpreted your question.

4

Correct & enrich

Improve responses by fixing synonyms, adding supplemental instructions, and giving the model more context about the data — then re-test.
