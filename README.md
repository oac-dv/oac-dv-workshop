# Oracle Analytics Cloud DV Workshop

Customer-facing hands-on workshop page for the Oracle Analytics Cloud Data Visualization session.

## GitHub Pages

This repository is prepared to publish from the `docs/` folder.

After pushing to GitHub:

1. Open the repository on GitHub.
2. Go to **Settings** > **Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and the `/docs` folder.
5. Save the setting and wait for GitHub Pages to publish the site.

The published page will use `docs/index.html`.
